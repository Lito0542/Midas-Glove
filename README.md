# Midas Glove
* #### 역할 : VR 컨텐츠 개발
* #### 개발 언어, 툴 : C#, Unity

## 프로젝트 소개
### 게임 소개
* 장갑형 VR 컨트롤러를 테스트해 볼 수 있는 프로그램입니다.
### 게임 화면
* **게임의 메인화면입니다. 인게임에서의 닉네임을 입력할 수 있습니다.**
+ **개발시 난관**
   + 아두이노 기기와 따로 시리얼 통신을 해야할 필요가 있었음
      + SerialManager클래스를 생성하여 시리얼 통신을 담당하도록 하였습니다.
